ref和reactive的差别

ref---》基本数据类型
reactive----》对象类型类型

ref

ref针对的是基础数据类型

ref 将变量变成可以浏览器实时渲染，等于获取网页文档标签元素对象，对对象的值进行修改

import ref from 'vue'
let people = ref('zhansan')

ref函数将people从字符串变成对象
在后续的对people的使用，要用people.value对值的获取
但在html里面变量直接使用people对象。
这个人名字：{{people}}

rective

将对象变量(数组也是)变成可以浏览器实时渲染的函数
let people = reactive({
id : 123,
name : 'zhangsan'
})
